검색 예시 3가지
1. 배열검색
2. 선형 리스트 검색
3. 이진검색트리 검색

### 배열 검색 에서 활용하는 알고리즘
1. 선형 검색: 무작위로 늘어놓은 데이터 모임에서 검색 수행
2. 이진 검색: 일정한 규칙으로 늘어놓은 데이터 모임에서 빠른 검색 수행
3. 해시법: 추가, 삭제가 자주 일어나는 데이터 모임에서 빠른 검색 수행


### 선형 검색에서의 보초법(Sentinel)
* 선형 검색에서 종료 조건은 두 가지이다.
  1. 검색할 값을 발견하지 못하고 배열 끝을 지난 경우
  2. 검색할 값과 같은 요소를 발견한 경우
* 보초법은 1번째 연산을 없애서 비용을 반으로 줄일 수 있다.
  * 배열의 끝에 찾고자 하는 값을 추가하여 검색할 값을 반드시 발견하도록 하는 것이다.
  * 이때 끝에 추가하는 값을 보초라고 한다.

### 이진 검색
* 배열이 정렬되어 있어야 한다. 
* 시간 복잡도는 O(log n)
* 자바의 Arrays.binarySearch() 활용
