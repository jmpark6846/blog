  • guard 문
    • if 처럼, bool expression에 따라 실행할지 말지 결정할때 사용하는 statement이다.
    • if 와 반대로 거짓일 때 실행된다.
    • if나 if let을 사용하다보면 조건을 명시하기가 까다롭거나 코드 블럭이 여러번 중첩되는 경우가 있는데, 이를 해결하기 위해 guard를 사용한다.
