### 자바 애플리케이션에서 유효성 검사
* 자바에선 Bean Validation 이라는 데이터 유효성 검사 프레임워크 제공함
* Bean Validation은 데이터 검증 로직을 DTO같은 도메인 모델과 묶어서 각 계층에서 사용. 즉 검증 자체를 도메인 모델에 얹는 방식.
* 이를 위해 어노테이션을 사용한다
* 스프링 부트에서는 Bean Validation 명세를 구현한 구현체인 Hibernate Validator 를 표준으로 채택했다.


### 스프링부트의 유효성 검사
* 유효성 검사는 각 계층으로 데이터가 넘어오는 시점에 해당 데이터에 대한 검사 실시
* 스프링부트에서는 데이터 전송 시 DTO를 사용하기 때문에 DTO 객체를 대상으로 수행
* 상황에 따라 검사하기 위해 ValidationGroup으로 나눠 지정할 수 있다.
